This plugin adds automatic identifier generation to the "Create
Accession" form.  The form will default to an identifier such as:

  YYYY NNN

Where YYYY is the current year, and NNN is a 3-digit number.

To install, just activate the plugin in your config/config.rb file by
including an entry such as:

     AppConfig[:plugins] = ['generate_accession_identifiers']

Numbering is generated by checking existing accession identifiers in the
database for the current year and selecting the first available value,
starting from 001.  This avoids consuming numbers when users refresh
the accession create page.
